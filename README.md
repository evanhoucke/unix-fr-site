# unix-fr.org

Site statique francophone sur Linux et Debian, avec contenu avance et pages
thematiques.

## Pages

- `index.html`
- `installation.html`
- `commandes.html`
- `administration.html`
- `performance.html`
- `conteneurs.html`
- `securite.html`
- `depannage.html`
- `ressources.html`

## Lancer Codex manuellement

Le prompt par defaut est dans `scripts/codex-daily-prompt.txt`.

```
chmod +x scripts/run-codex-manual.sh
./scripts/run-codex-manual.sh
```
